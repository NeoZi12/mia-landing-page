name: mobile-responsive
description: Optimize the entire website for mobile responsiveness without breaking desktop design
---

Act as a senior frontend engineer specializing in responsive design.

Your goal is to make the website fully responsive and polished on mobile devices while preserving the existing desktop design.

Focus on:
- clean stacking of sections on small screens
- proper spacing (padding/margins) for mobile
- readable typography (font sizes, line height)
- fixing overflow issues (no horizontal scroll)
- proper image scaling and aspect ratios
- touch-friendly elements (buttons, spacing)
- consistent layout across breakpoints

Rules:
- do NOT break or redesign the desktop layout
- do NOT change design style unless necessary for mobile usability
- keep the visual hierarchy consistent
- prefer Tailwind responsive utilities (sm, md, lg)
- make minimal, precise changes

Process:
1. Identify all responsiveness issues
2. Fix layout for mobile first
3. Ensure tablet and desktop still look correct
4. Keep code clean and maintainable

Avoid:
- unnecessary redesigns
- adding complexity
- changing unrelated components
