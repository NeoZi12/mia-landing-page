Optimize the frontend for performance.

Focus on:
- reducing unnecessary client-side JavaScript
- optimizing images and assets
- avoiding heavy libraries unless needed
- improving loading speed
- keeping the implementation lightweight
- preserving visual quality while improving performance

Avoid:
- unnecessary dependencies
- large unoptimized images
- overengineering
