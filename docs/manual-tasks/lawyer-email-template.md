## לפני השליחה — צ׳קליסט קצר למיה

- [ ] מצאי עו״ד מתאים. המלצות:
  - עו״ד שמכיר/ה את המשרד אישית (הסכמי לקוחות, דיני עבודה).
  - ואם אין — חפשי "דיני פרטיות" או "עו״ד לבעלי עסקים" באזור הצפון.
  - אפשרות זולה: **MyLawyer.co.il**, **LegalZoom ישראל**, **אדלפי** — סקירות מסמכים במחיר ₪250-₪500.
- [ ] ייצאי את שלושת דפי האתר ל-PDF. פתחי `/privacy`, `/terms`, `/accessibility` בדפדפן → Ctrl+P → שמירה כ-PDF.
- [ ] צרפי גם את הקובץ `docs/manual-tasks/lawyer-review-checklist.md` (אפשר לשלוח כ-PDF או להדפיס).
- [ ] התאימי את תבנית המייל: החליפי `[שם עו״ד]`, ואם יש תאריך יעד ספציפי ציינית אותו.
- [ ] שלחי.

## כשהתגובה חוזרת

- העבירי לנעם את המסמך המתוקן + התשובות לצ׳קליסט.
- נעם יטמיע את התיקונים בקוד האתר (משימה P2.8 בתכנית).
- תוך 1-2 ימי עבודה — שלושת הדפים יעלו לאתר.
