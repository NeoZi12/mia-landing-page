# הגדרת כתובת דוא״ל לפניות פרטיות — משימה למיה

> **Purpose:** Mia needs a dedicated email address (published in the Privacy Policy and Accessibility Statement) where users can send privacy requests and accessibility complaints. Output: a working email address, pasted into `src/lib/siteConstants.ts` as `PRIVACY_CONTACT_EMAIL`.

---

## למה צריך כתובת דוא״ל ייעודית

- **תיקון 13 לחוק הגנת הפרטיות** מחייב ערוץ קשר נגיש לפניות בענייני פרטיות (זכות עיון, תיקון, מחיקה).
- **תקנות הנגישות** מחייבות רכז/ת נגישות שאיש קשר עמו/עמה מצוין בהצהרה.
- **נוחות פנימית** — הפרדה בין פניות משפטיות לפניות שירות רגילות עוזרת לתעדוף ולתיעוד.

כרגע, הקונסטנטה `PRIVACY_CONTACT_EMAIL` ב-`src/lib/siteConstants.ts` מוגדרת כ-**`privacy@mia-tax.co.il`** כ-placeholder. צריך ליצור את הכתובת הזו (או להחליף בכתובת אחרת שעובדת).

## אופציה A — אליאס חדש (מומלץ)

יצירת כתובת **`privacy@mia-tax.co.il`** שמעבירה את כל ההודעות לתיבה הראשית של מיה.

### יתרונות
- כתובת מקצועית וייעודית.
- קל לזהות מיידית שמדובר בפנייה משפטית.
- לא משנה את התיבה הקיימת — ההודעות עדיין מגיעות למיה.

### איך עושים זאת

תלוי בנותן השירות של הדומיין `mia-tax.co.il`:

**אם הדומיין אצל Google Workspace:**
1. היכנס ל-Google Admin Console → Users → בחר את המשתמשת של מיה.
2. Account → Aliases → Add an alias → `privacy`.
3. שמור.

**אם הדומיין אצל Microsoft 365:**
1. Admin Center → Users → Active users → מיה.
2. Manage email aliases → Add an alias → `privacy@mia-tax.co.il`.

**אם הדומיין אצל ספק אחר (Rackspace / Zoho / בלוגר / Namecheap):**
1. היכנס/י ל-cPanel או לממשק הניהול.
2. חפש/י "Email forwarders" או "Email aliases".
3. צרי forwarder: `privacy@mia-tax.co.il` → `<כתובת הדוא״ל הראשית של מיה>`.

**אם אין לדומיין שירות דוא״ל פעיל כלל:**
- **פתרון זול:** Zoho Mail Free (https://www.zoho.com/mail/) — עד 5 משתמשים חינם, כולל דומיין מותאם. 30-60 דקות הגדרה (רשומות MX ב-DNS).
- **פתרון פרימיום:** Google Workspace / Microsoft 365 — ~$6/חודש.

### תוצר
כתובת פעילה `privacy@mia-tax.co.il` שמיה מקבלת אליה הודעות. הודיעי לנעם שהכתובת עובדת — אין צורך לעדכן קוד אם הכתובת כבר מוגדרת בקוד.
